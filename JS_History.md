### 📖 JavaScript ki Kahani

1995 ka time tha. Web duniya abhi bachpan mein thi, aur browsers ka ek naya star chamak raha tha — Netscape Navigator. Us time internet pages mostly static hote the — matlab sirf text aur images, koi real interaction nahi.

#### 🚀 Interactive web ki zarurat

Netscape Navigator ki team ne socha:
“Sirf static pages se kaam nahi chalega… hume web ko interactive banana hoga.”

Isi idea se ek programming language banane ka plan bana.

#### 👨‍💻 Brendan Eich ka 10 din ka miracle

Netscape ne ek engineer ko ye task diya — Brendan Eich.

Unhone sirf **10 din** ke andar ek nayi language bana di. Is language ka naam pehle rakha gaya **Mocha**, phir badal kar **LiveScript** kar diya gaya.

#### ☕ Java ka hype aur naam ka twist

Us time ek aur language bahut popular ho rahi thi — Java.

Marketing ke chakkar mein LiveScript ka naam badal kar rakh diya gaya:
👉 **JavaScript**

Important baat:
Java aur JavaScript ka **koi direct relation nahi hai** — dono completely different languages hain.

#### 🌐 Microsoft ka entry aur JScript

1996 mein Microsoft ne apna browser launch kiya — Internet Explorer.

Microsoft ne dekha ki JavaScript popular ho raha hai, to unhone apna version bana diya:
👉 **JScript**

Matlab us time:

* Netscape → JavaScript
* Internet Explorer → JScript

Dono browsers apne-apne features add kar rahe the, jiski wajah se websites kabhi ek browser mein chalti, kabhi dusre mein break ho jaati thi.

#### ⚠️ Big problem

Developers ke liye sabse badi problem thi:
“Ek website banani hai to alag-alag browsers ke liye alag code likhna padega!”

Is confusion ko solve karne ke liye standardization ki zarurat padi.

#### 🏛️ ECMAScript ka birth

Is problem ko handle karne ke liye language ko standard banane ka kaam diya gaya ek organization ko — Ecma International.

Unhone JavaScript ka standard define kiya jise bola gaya:
👉 ECMAScript

Is standard ko ECMA-262 document ke through define kiya gaya.

#### 🧠 TC39 — future decide karne wali committee

ECMAScript ko maintain karta hai ek expert group:
👉 **TC39 (Technical Committee 39)**

Ye log decide karte hain:

* JavaScript mein kaunse naye features aayenge
* kaunse features safe hain
* aur language ka future direction kya hoga

#### 📅 Versions ka evolution

* **ES1 (1997)** → first official standard
* **ES5 (2009)** → major improvements
* **ES6 (2015)** → biggest revolution (Modern JavaScript)
* ES6 ke baad se JavaScript ko “Modern JavaScript” ka base mila

Aur uske baad:

* ES7 (2016)
* ES8 (2017)
* aur har saal naye updates aane lage

#### 🎯 Final reality

Aaj jo JavaScript hum use karte hain:

* wo ek browser war se nikla hua language hai
* jise quickly banaya gaya
* phir standard banaya gaya
* aur aaj duniya ki sabse important web language ban chuka hai
